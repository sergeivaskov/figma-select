# Select Similar Layers

Плагин для Figma, который позволяет находить и выделять слои с похожей структурной позицией в дереве документа.

## Описание

Плагин анализирует структурное расположение выбранного слоя в дереве Figma и находит все слои с аналогичной позицией на странице. Это полезно для массового выделения повторяющихся элементов интерфейса.

## Установка

1. Клонируйте репозиторий:
```bash
git clone git@github.com:sergeivaskov/figma-select.git
cd figma-select
```

2. Установите зависимости:
```bash
npm install
```

3. Соберите плагин:
```bash
npm run build
```

4. В Figma: `Plugins` → `Development` → `Import plugin from manifest` → выберите `manifest.json`

## Использование

1. **Select Similar**: Выделите один слой в Figma, запустите команду, и все похожие слои будут автоматически выделены.
2. **Select First/Last Child**: Выделяет первый или последний дочерний элемент внутри выбранного контейнера.
3. **Hide all instances**: Скрывает все инстансы выбранного мастер-компонента или инстанса.
4. **Select Random**: Оставляет выделенным случайное количество объектов из изначально выделенных (полезно для создания хаотичных паттернов).

## Разработка

```bash
npm run watch  # Автоматическая пересборка при изменениях
```
