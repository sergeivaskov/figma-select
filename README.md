# Select Similar Layers

Плагин для Figma, который позволяет находить и выделять слои с похожей структурной позицией в дереве документа.

## Описание

Плагин анализирует структурное расположение выбранного слоя в дереве Figma и находит все слои с аналогичной позицией на странице. Это полезно для массового выделения повторяющихся элементов интерфейса.

## Установка

1. Клонируйте репозиторий:
```bash
git clone git@github.com:sergeivaskov/figma-select.git
cd figma-select
```

2. Установите зависимости:
```bash
npm install
```

3. Соберите плагин:
```bash
npm run build
```

4. В Figma: `Plugins` → `Development` → `Import plugin from manifest` → выберите `manifest.json`

## Использование

1. Выделите один слой в Figma
2. Запустите плагин
3. Все похожие слои будут автоматически выделены

## Разработка

```bash
npm run watch  # Автоматическая пересборка при изменениях
```
